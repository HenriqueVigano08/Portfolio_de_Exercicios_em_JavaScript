# Lista de Exercicios

Este repositório contém uma lista de exercícios práticos desenvolvidos com **HTML** e **JavaScript**, com o objetivo de treinar lógica de programação, manipulação de dados e interação com o usuário por meio de páginas web simples.

## 📌 Sobre o projeto

O projeto foi criado para praticar conceitos fundamentais de programação utilizando JavaScript, como:

- Variáveis
- Condicionais
- Estruturas de repetição
- Funções
- Operações matemáticas
- Manipulação do DOM
- Validação de dados

Cada exercício possui uma página HTML própria e um arquivo JavaScript responsável pela lógica da aplicação.

## 🚀 Tecnologias utilizadas

- HTML5
- JavaScript
